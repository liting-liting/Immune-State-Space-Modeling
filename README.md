# Immune-State-Space-Modeling
# TCR Immune Repertoire Dynamics Analysis

This repository contains code for analyzing T cell receptor (TCR) repertoire dynamics using a state-space modeling framework.

## Features

- PCA-based immune state-space visualization
- State radius (intra-individual variability)
- Temporal step distance (dynamic change)
- Stability index
- Effect size (Cohen's d)

## Requirements

- R (>=4.0)
- ggplot2
- dplyr
- ggpubr
- effsize

## Usage

source("scripts/TCR_dynamics_analysis.R")
